Detailed Description

Entry point for the Container Monitor application.

Initializes configuration, logging, message queues, database, resource threads, event listeners/processors, UI components, and manages graceful shutdown.

int main ( int  argc,
char *  argv[] 
)

Main entry point for the Container Monitor application.

  • Parses configuration and initializes logging.
  • Sets up message queues and signal handlers.
  • Initializes database and resource thread pool.
  • Starts event listener, processor, resource monitor, and UI components.
  • Waits for shutdown signal and performs graceful cleanup.
